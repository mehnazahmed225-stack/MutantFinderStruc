# MutantFinderStruc

This is an app that works with openclaw to analyze results from a program that finds all possible missence mutations in a protein. It will point you in the direction of interesting missence mutations to study as well as analyze the resistance to mutation of any given protein. It's a program build with Ryrosetta, Pyside6 and Openclaw. 

Why you'd use this product is to find missence mutations that are structurally interesting.
